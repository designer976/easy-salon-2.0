Estes Termos regulam o uso da plataforma Easy Salon, que oferece soluções digitais de:

Política de Privacidade

1. COMPROMISSO COM A PRIVACIDADE

A Easy Salon respeita a privacidade dos usuários e protege seus dados pessoais conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

2. DADOS COLETADOS

Podemos coletar:

  • Nome
  • E-mail
  • Telefone
  • Dados do estabelecimento
  • Dados de clientes cadastrados pelo usuário
  • Informações de uso da plataforma
  • Endereço IP e dados de navegação

3. FINALIDADE DO USO DOS DADOS

Os dados são utilizados para:

  • Operar e melhorar a plataforma
  • Permitir agendamentos e comunicações
  • Cumprir obrigações legais
  • Enviar comunicações operacionais
  • Análises internas de performance e segurança

4. DADOS DE CLIENTES DOS USUÁRIOS

O usuário é o controlador dos dados de seus clientes.

A Easy Salon atua como operadora, tratando os dados apenas conforme instruções do usuário.

5. COMPARTILHAMENTO DE DADOS

Os dados poderão ser compartilhados:

  • Com fornecedores de tecnologia e hospedagem
  • Para cumprimento de obrigações legais
  • Mediante consentimento do titular

Nunca vendemos dados pessoais.

6. ARMAZENAMENTO E SEGURANÇA

Adotamos medidas técnicas e organizacionais para proteger os dados contra:

  • Acessos não autorizados
  • Vazamentos
  • Alterações indevidas

7. DIREITOS DOS TITULARES

O titular pode solicitar:

  • Confirmação de tratamento
  • Correção
  • Exclusão
  • Revogação de consentimento

Solicitações podem ser feitas pelo canal de contato da plataforma.

8. COOKIES

Utilizamos cookies para:

  • Melhorar a experiência do usuário
  • Análises estatísticas
  • Funcionalidades essenciais

O usuário pode gerenciar cookies no navegador.

9. RETENÇÃO DOS DADOS

Os dados são mantidos:

  • Pelo tempo necessário à finalidade
  • Para cumprimento de obrigações legais
  • Até solicitação de exclusão, quando aplicável

10. ALTERAÇÕES DESTA POLÍTICA

Esta Política pode ser atualizada periodicamente.

11. CONTATO

Em caso de dúvidas sobre privacidade e dados pessoais, o usuário pode entrar em contato pelos canais oficiais da Easy Salon.
